Assignment 2: Variables

Description

For this assignment we had to do our self portrait again but this time using variables. I decided to pick one of my favorite activities to use for the assignment instead. My favorite activity involved playing video games. We had to use variables this time instead of hardcoded values to get used to on how p5 Javascript works.

The first thing my friends pointed out when seeing this assignment was the pixel art on the TV. I was questioned about it if I used an image or found a code to it online and pasted it onto my sketch. I did the image with just rects and used the image on the side as reference, I didn't do the full character since it would have taken me a while to finish. As for the the library of games, I used my PS4, my games and TV as reference and made them with rects, an ellipse and a quad. Since I wanted to put in text on the spine of the games, I went ahead and learned how to use the text and rotate function to make the assignment look like the way I wanted it to look. Out of everything on this sketch, the pixel art took me the longest to do.
